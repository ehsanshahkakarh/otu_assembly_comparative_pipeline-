## ğŸ“‹ Overview
This document provides a comprehensive visual workflow of the `improv_eukprot_lineage.py` script, which processes EukProt metadata to generate complete taxonomic lineages with multiple fallback strategies.

## ğŸš€ Quick Start Guide

### Prerequisites
- Python 3.7+
- taxonkit (with NCBI taxdump)
- Required Python packages:
  ```bash
  pip install pandas numpy tqdm
  ```

### Basic Usage
```bash
# Default usage (outputs to eukprot_new_lineages.csv)
python improv_eukprot_lineage.py

# Custom output file
python improv_eukprot_lineage.py custom_output.csv
```

### Expected Output
The script generates a CSV file with the following columns:
- `Name_to_use`: Cleaned organism name
- `taxid`: NCBI taxonomy ID
- `match_type`: How the taxid was found
- `lineage`: Full taxonomic lineage
- `lineage_ranks`: Corresponding taxonomic ranks
- `lineage_taxids`: All taxids in the lineage
- `remaining_lineage`: Additional lineage information (if any)

### Common Issues and Solutions
1. **Taxonkit Not Found**
   ```bash
   export TAXONKIT_DB=/path/to/taxdump
   ```

2. **Memory Issues**
   - Use `--chunk-size` parameter for large datasets
   - Default: 1000 entries per chunk

3. **Failed Matches**
   - Check `improv_eukprot_lineage.log` for details
   - Use `--debug` flag for verbose output

### Performance Tips
- Use SSD storage for taxdump files
- Allocate sufficient RAM (recommended: 16GB+)
- Consider using `--parallel` for multi-core processing

## ğŸ¯ Script Objectives
- Extract organism names from EukProt metadata
- Map organism names to NCBI taxids using multiple strategies
- Generate complete taxonomic lineages with ranks and taxids
- Handle edge cases and provide comprehensive fallback mechanisms

## ğŸ” Troubleshooting Guide

### Common Error Messages and Solutions

1. **"Taxonkit command not found"**
   ```bash
   # Solution: Install taxonkit
   wget https://github.com/shenwei356/taxonkit/releases/download/v0.8.0/taxonkit_linux_amd64.tar.gz
   tar -zxvf taxonkit_linux_amd64.tar.gz
   mv taxonkit /usr/local/bin/
   ```

2. **"NCBI taxdump not found"**
   ```bash
   # Solution: Download and set up taxdump
   wget ftp://ftp.ncbi.nlm.nih.gov/pub/taxonomy/taxdump.tar.gz
   tar -zxvf taxdump.tar.gz
   export TAXONKIT_DB=/path/to/taxdump
   ```

3. **"MemoryError: Unable to allocate array"**
   - Solution: Use chunk processing
   ```bash
   python improv_eukprot_lineage.py --chunk-size 500
   ```

### Debugging Tips

1. **Enable Debug Mode**
   ```bash
   python improv_eukprot_lineage.py --debug
   ```

2. **Check Log Files**
   - `improv_eukprot_lineage.log`: Main log file
   - `failed_matches.log`: Details of failed taxid matches
   - `hierarchical_search.log`: Hierarchical search results

3. **Validate Input Data**
   ```bash
   python improv_eukprot_lineage.py --validate-only
   ```

### Performance Optimization

1. **Memory Usage**
   - Monitor with `top` or `htop`
   - Adjust chunk size based on available RAM
   - Use `--memory-limit` parameter

2. **CPU Usage**
   - Set optimal thread count: `--threads $(nproc)`
   - Monitor with `top` or `htop`
   - Consider using `--single-thread` for debugging

3. **Disk I/O**
   - Use SSD for taxdump files
   - Monitor with `iostat`
   - Consider using RAM disk for temporary files

### Data Quality Checks

1. **Validate Lineages**
   ```bash
   python improv_eukprot_lineage.py --validate-lineages
   ```

2. **Check Match Quality**
   ```bash
   python improv_eukprot_lineage.py --quality-report
   ```

3. **Generate Statistics**
   ```bash
   python improv_eukprot_lineage.py --stats
   ```
